# Cluedo (Clue) - Python Murder Mystery

A digital implementation of the classic board game Cluedo. Players make suggestions and use logic to deduce the Murderer, Weapon, and Room.

## Prerequisites

- Python 3.8 (or higher)
- No external libraries required

## How to Play

1. Clone the repository

```python
git clone https://github.com/EmmanuelChiobi/CS670_Cluedo.git
```

2. Navigate to the "CS670_Cluedo" directory

```python
cd CS670_Cluedo/
```

3. Run the game

```python
python main.py
```
